# ComfyUI Style Prompts Collection

A comprehensive collection of artistic style prompts for use with ComfyUI, particularly with nodes that support style-based prompting. This repository contains meticulously organized style definitions that can enhance your AI image generation workflows. Works excellently with nodes from WAS Node Suite (https://github.com/WASasquatch/was-node-suite-comfyui), providing seamless integration for style-based image generation.

## Overview

This collection provides a wide variety of artistic styles, techniques, and aesthetics that can be used as prompts in AI image generation tools. Each style includes both positive and negative prompts to guide the AI toward producing images that match the desired aesthetic while avoiding unwanted elements.

## Contents

The repository includes:

- **styles-organized.csv** - Main collection in CSV format
- **styles-organized.json** - Same collection in JSON format
- **styles.csv** - Original collection in CSV format
- **styles.json** - Original collection in JSON format
- **styles-new-unsorted.csv** - Additional unsorted styles

## Usage

### With ComfyUI
These style prompts can be used with ComfyUI nodes that support style-based prompting, such as:
- WAS Node Suite (https://github.com/WASasquatch/was-node-suite-comfyui) - works excellently with these style definitions
- Other custom nodes that accept style presets
- Any node that processes text prompts with predefined styles


### CSV Format
The CSV files contain three columns:
- `name`: The style category and name
- `prompt`: Positive prompt elements for the style
- `negative_prompt`: Elements to avoid in the generated image

### JSON Format
The JSON files contain the same information in a structured format:
```json
{
  "Style Name": {
    "prompt": "positive prompt elements...",
    "negative_prompt": "negative prompt elements..."
  }
}
```
